# 🐱 Чанкер РА - Веб версия

Профессиональный адаптер баз знаний с умным чанкингом для подготовки данных к RAG (Retrieval-Augmented Generation).

## 🚀 Описание

Веб-приложение для интеллектуальной обработки структурированных данных:
- Загрузка файлов в форматах JSON, TXT, DOCX, PDF
- Умное распознавание структуры данных
- Настройка полей для чанкинга
- Разделение на чанки с различными стратегиями
- Экспорт результатов в JSON, TXT, DOCX, PDF

## 🌟 Особенности

- **🔐 Регистрация и авторизация пользователей**
- **⚠️ Система логирования всех действий** (для отслеживания NDA данных)
- **🧠 Умный парсинг** JSON из текстовых файлов и PDF
- **⚙️ Настраиваемый чанкинг** с тремя стратегиями:
  - 🏆 Универсальный (для векторного поиска)
  - 🤖 Speech Sense (строгий парсинг)
  - 📄 Плоский текст (ключевые слова)
- **📤 Экспорт в 4 форматах** с двумя режимами:
  - 👁️ Для чтения (красивый документ)
  - 🤖 Для Базы Знаний (чистый JSON)
- **👥 Администрирование пользователей**

## 🛠 Технологии

- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript, Bootstrap 5
- **База данных**: SQLite
- **Обработка файлов**: pdfplumber, python-docx, reportlab
- **Чанкинг**: langchain-text-splitters
